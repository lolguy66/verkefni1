Insurgency Sandstorm

Insurgency Sandstorm er 'AAA' leikur sem að kom út 12 des. 2018. Höfundur leiksins var: 'New World Interactive' og var framleiddur af 'Focus Home Interactive'. Insurgency Sandstorm er lið byggður, markimað stilltur, nútíma stríðsleikur. Leikurinn er mest spilaður af fullorðnu fólki (27 -39) mest karlmenn og til að geta njótt leiksins þarftu í mesta lagi vera með áhuga á hernaðarlegu efni.
Leikurinn hefur þróast með því það hafa bæst í hann,

  1. 1. Sérsniðnar persónustillingar
  2. 2. Ökutæki
  3. 3. Ný 'möpp'
  4. 4. Ný 'Unreal 4' leikjavél
  5. 5. Og miklu fleiri byssur
Og miklu meira.
Síðan að 'Unreal 4' leikjavélin kom í leikinn þá hefur búnaðurinn þróast með því. Með þessari uppfærslu hefur Raunsæinn verið marktækt betur og grafík miklu, miklu betri.
"Varðandi framtíðarhorfu leiksins, fólk sem spilar leikinn okkar vil ný 'möpp' og meiri raunveruleikari byssur. Þannig við reynum að bæta því í hverri uppfærslu. Þau vilja fleiri 'Gamemodes', 'Sérsniðna miðlara' og önnur fyribæri til að gera leikinn betri. Þannig að við gerum okkar besta til að bæta því inn." segir Alex Blonski, einn af framleiðslu mönnunum leiksins í viðtali.
